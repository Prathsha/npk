## Add Photos

1. Put your pictures in `public/photos`.
2. Open `public/photos/photos.json`.
3. Add each file name and caption.

Example:

```json
[
  { "src": "photo1.jpg", "caption": "The day everything started." },
  { "src": "photo2.png", "caption": "My favorite smile." }
]
```

## Run

```bash
pnpm install
pnpm dev
```

Then open the local URL shown in the terminal.
